# cnh5-pg-archive

## 项目简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态 HTML 文件形式存在，不依赖于任何特定域名或具体网站，仅作为独立页面资源进行整理与公开。

仓库遵循“一个页面一个文件”或“一个页面一个文件夹”的归档方式，便于直接浏览、引用或二次分发。

## 目录结构说明

```
cnh5-pg-archive/
├── README.md          # 本文件
├── index.html         # 可选：仓库首页或导航页
├── page-001/          # 示例：第一个归档页面目录
│   └── index.html
├── page-002/          # 示例：第二个归档页面目录
│   └── index.html
└── ...                # 更多归档页面
```

- 每个归档页面通常存放在独立文件夹内，文件夹名称以 `page-` 开头并附带序号或标识。
- 如果页面非常简单，也可以直接以单文件形式放在仓库根目录。
- 根目录下的 `index.html` 可作为导航页，列出所有归档页面的链接。

## 页面归档说明

- 本仓库收录的 HTML 页面均为独立、完整的静态页面，不依赖外部后端服务。
- 页面内容可能包含文本、图片、样式和脚本，所有资源尽量内联或随页面一同归档。
- 归档页面不保证实时更新，也不承诺长期可用性。
- 页面可能来自不同时期、不同来源，风格和内容各有差异。

## 维护说明

- 本仓库由维护者不定期更新，新增或删除归档页面。
- 欢迎通过 Issue 或 Pull Request 提交新的归档页面或修正建议。
- 提交的页面应保持独立、无恶意代码、不侵犯他人权益。
- 维护者保留对页面归档与否的最终决定权。

## 使用方式

- 可直接通过 GitHub Pages 或任何静态文件服务器浏览本仓库中的 HTML 页面。
- 也可以将仓库克隆到本地，在本地环境中打开 HTML 文件查看。
- 引用或分发时，请保留页面原有署名（如有）。
